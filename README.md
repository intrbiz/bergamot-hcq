Bergamot Monitoring HCQ
=======================
Bergamot Monitoring HCQ is a simple in memory queue broker based on Hazelcast.  
It leverages Hazelcasts distributed data structures and adds a Websocket based 
protocol layer to setup queues, exchange and publish, consume.

It is designed to be a key component of the Bergamot Monitoring project.

See https://bergamot-monitoring.org/ for more information on the project.
